[ 피그마 컴포넌트 ]

-> 만들고자 하는 요소나 그룹을 선택 후 상단의 컴포넌트 버튼을 클릭하거나
ctrl+alt+K 단축키로 만든다!
레이어에서 이동도 가능하다!

1. 일반 그룹화와는 다른 재사용성을 위한 공용모듈화 요소
2. 원본개념이 있어서 원본을 수정하면 카피버전은 자동으로 수정된다!
3. 원본은 채워진 다이아몬드
   카피버전은 비워진 다이아몬드

4. 일반적으로 컴포넌트는 프레임 바깥쪽에 만들고 관리하는 것이 좋다!

5. 제작된 컴포넌트는 Assets 메뉴에 있다!
  -> 또는 상단 메뉴에 Resources 버튼 아래에도 컴포넌트를 볼 수 있다!


































[ 피그마 컴포넌트 배너1 ]

1. 박스만들고 이미지넣기
2. 컴포넌트로 변경
3. Variants 버튼으로 내부 컴포넌트 만들기
4. 반복하여 4개 생성
5. Prototype 메뉴에서 각 내부 컴포넌트를 아래로 연결
6. 각 속성 이벤트는 After Delay 를 선택
    (여기서 시간은 1/1000초임 - 2000ms는 2초)
7. 애니메이션 옵션은 Dissolve 선택
    (이징가속도 : ease-in-out 처음과 끝에 천천히)
8. Assets 또는 Resources에서 해당 컴포넌트를
    드래그 하여 원하는 프레임에 넣기
9. Present 화면 띄우기(ctrl+alt+엔터)

※ 만약 배너가 안움직이면 로딩시간이 안맞아서
그럴 수 있으므로 오른쪽 아래 Restart 버튼을 클릭한다!
